# UK Housing Market Investment Analysis
End-to-end data analytics project analysing UK house prices (2010–2024). Cleaned and transformed UK Land Registry data, calculated annual and cumulative growth metrics, and built an interactive dashboard to identify top-performing investment regions.

## Tools
- Python
- Power BI
